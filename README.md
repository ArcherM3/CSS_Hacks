# CampusFix 
### Team: WebWizards

---

## Problem
Students face delays in complaint resolution due to lack of a centralized system.

---

## Solution
CampusFix is a smart complaint management platform that allows students to submit complaints and track their resolution in real time.

---

## Features
- Complaint submission system  
- Role-based dashboards (Student & Admin)  
- Department-wise routing  
- Status tracking (Pending, In Progress, Resolved)  
- Admin dashboard with complaint management  

---

## Workflow
Student → Submit Complaint → Routed to Department → Admin → Resolved  

---

## Tech Stack
- HTML  
- CSS
- JS

---

## Future Scope
- AI-based complaint routing  
- Mobile application  
- Notification system
